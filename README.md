# ds-480-bias-paper

This is a repository containing the data analysis that I did for my DS 480 bias paper on bilingual aphasia. It is split into 3 files: one for my primary analysis (which was a linear mixed-effects model looking at
the relationship between WAB-R sequential command task performance and lifestyle factors), one for my exploratory analysis (Pearson's r correlating lifestyle variables with sequential command scores across language), and the final file contains the code I used to create my heatmap visuals. The primary analysis was done in MATLAB, but I switched to R for the heatmap creation and ended up using R for the exploratory analysis as well.
